Database Management System's Basic Terminology

Data

Data is a collection of facts, such as numbers, words, measurements, observations, or even just descriptions of things. It can be structured, semi-structured, or unstructured, and is the raw input used for processing information.

Information

Information is processed data that is meaningful and useful. It provides context for data and allows for better decision making and understanding of the world.

Meta Data

Metadata is data that describes other data. It provides information about a certain item's content, context, quality, condition, and other characteristics.

Data Dictionary

A data dictionary is a centralized repository of information about data, such as meaning, relationships to other data, origin, usage, and format. It helps in managing and controlling data resources.

Table

A table is a collection of related data held in a structured format within a database. It consists of columns and rows. Each row is a unique instance of data, and each column holds a different attribute of the data.

Tuple

A tuple is a single row in a table, which represents a single record of data. It is a finite ordered list of elements.

Domain

A domain is the set of allowable values for a given attribute, defining the possible values that can be assigned to a column in a table.

Key

A key is an attribute or a set of attributes that uniquely identifies a tuple within a table. It can be a primary key, candidate key, or foreign key.
